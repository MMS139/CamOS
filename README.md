# CamOS
Welcome to CamOS! This is a custom OS I (CamoKid) coded as part of Hack Club's Stardance program. This took me almost 15 hours, so please take your time exploring and I hope you enojoy!

---

## Built-In Apps

| Application | Description |
| --- | --- |
| Welcome | Basic introduction page to the OS |
| Terminal | Fake terminal shell with 6 commands, including `-clear` & `-about` (see [Terminal Commands](#terminal-commands) below) |
| Calculator | Simple calculator with 4 basic operations and exponents |
| Settings | Options to change wallpaper and brightness |
| Music Player | Loaded with 7 cool songs, including Alexander Rybak - Fairytale, & Laufey - From the Start |

### Terminal Commands
```
-help              See list of all available commands
-about             Read about CamOS
-time [timezone]   Find current time across the world (currently limited to UTC & AEST)
-clear             Clear the terminal
-refresh           Refresh the page
-quit              Quit the terminal
```
